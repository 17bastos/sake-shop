# 🍶 Sake Shops App

This is a mobile application built as part of a technical assessment for a Mobile Developer role (Android). The app displays a list of local sake shops and provides detailed information about each one, including address, rating, website, and map location.

## 📱 Features

- List of local sake shops (name, address, rating)
- Detailed view of each shop:
    - Name
    - Description
    - Image
    - Star rating
    - Clickable address (opens in Google Maps)
    - Website link (opens in browser or custom tab)
